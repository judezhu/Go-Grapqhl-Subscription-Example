# Go-Grapqhl-Subscription-Example

## Run Client

1. `cd client`
2. `npm install` 
3. `npm start`
4. Connect VPN if needed. 
5. Open browser `deveoper tools` and check `network type`. 
6. The messages in the websocket connection network call.
